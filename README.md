# 🐍 贪吃蛇游戏

这是一个使用纯 HTML、CSS 和 JavaScript 开发的贪吃蛇网页游戏，非常适合初学者学习网页开发。

## 📁 项目文件说明

- `index.html` - 游戏的主页面结构
- `style.css` - 游戏的样式和美化
- `game.js` - 游戏的核心逻辑代码
- `README.md` - 本说明文件

## 🚀 如何运行游戏

### 方法一：直接在浏览器中打开（最简单）

1. 找到 `index.html` 文件（在 `/Users/castle/snake-game/` 文件夹中）
2. 双击文件，它会在你的默认浏览器中打开
3. 点击"开始游戏"按钮，使用方向键控制蛇的移动

### 方法二：使用 Cursor 的 Live Server（推荐）

如果你安装了 Live Server 扩展：

1. 在 Cursor 中右键点击 `index.html`
2. 选择 "Open with Live Server"
3. 游戏会在浏览器中自动打开，并且修改代码后会自动刷新

### 方法三：使用 Python 简单服务器（适合学习）

如果你安装了 Python：

1. 在终端中进入项目文件夹：
   ```bash
   cd ~/snake-game
   ```

2. 运行 Python 服务器：
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # 或者 Python 2
   python -m SimpleHTTPServer 8000
   ```

3. 在浏览器中打开：`http://localhost:8000`

## 🎮 游戏操作

- **方向键** ⬆️ ⬇️ ⬅️ ➡️：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **开始游戏**：点击按钮或按任意键开始
- **暂停**：暂停当前游戏
- **重置**：重新开始游戏

## 🎯 游戏规则

1. 控制蛇吃红色的食物
2. 每吃到一个食物，得分 +10，蛇会变长
3. 撞到墙壁或自己的身体，游戏结束
4. 尝试获得更高的分数！

## 🔧 如何修改游戏

### 改变游戏速度
在 `game.js` 文件中找到这一行：
```javascript
gameLoop = setInterval(gameMainLoop, 150);
```
将 `150` 改为其他数字（数字越小，游戏越快）

### 改变游戏大小
在 `game.js` 文件顶部修改：
```javascript
const GRID_SIZE = 20; // 每个格子的大小
const CANVAS_SIZE = 400; // 画布大小
```

### 改变颜色
在 `style.css` 和 `game.js` 中搜索颜色代码（如 `#667eea`），替换为你喜欢的颜色

## 🆘 遇到问题？

- 确保所有文件都在同一个文件夹中
- 确保浏览器支持 JavaScript
- 如果游戏不运行，检查浏览器的控制台是否有错误（按 F12 打开开发者工具）
